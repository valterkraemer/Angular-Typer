Introduction

Once upon a time I had to write an essay and had to use citing. I did not want to install Latex, don't get me wrong, I like Latex. But I did not want to install 2.4GB just to be able to make a couple of citations. So I thought: "Angular would work fine for this!". I can do citeing by typing {{'criteria' | cite}} = ({{'criteria' | cite}}). Sorry for incomplete documentation :/
